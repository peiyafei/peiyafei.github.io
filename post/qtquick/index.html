4.1 Hello Qt QuickApp再分析#

4.1.1 基本程序结构#

main.qml 的内容如下：

import QtQuick

Window {
    width: 640
    height: 480
    visible: true
    title: qsTr("Hello World")

    MouseArea {
        anchors.fill: parent
        onClicked: {
            Qt.quit();
        }
    }

    Text {
        id: txt1
        text: qsTr("Hello Qt Quick App")
        anchors.centerIn: parent
    }
}

一个QML文档由两部分组成

  • import 语句
  • QML 对象树
  1. import和C++中的#include类似，用来导入一个模块
  2. 一个QML文档就对应一颗以文档根对象为根的对象树
  3. 默认属性，

4.2 闲话Window#

GUI：Graphical User Interface 图形用户接口

CLI：Command Line User Interface 命令行用户接口

Window对象用于创建与操作系统相关的顶层窗口

4.2.1 Window 细说#

Wiindow在桌面上的位置，由x、y属性决定，大小由width、height属性决定

minimumWidth、minmumHeight 来限制窗口的最小尺寸；maximumWidth、maximumHeight 来限制窗口的最大尺寸

window对象的 visibility 属性用来设置窗口的状态，可取值如下：

  • 最小化窗口

    visibility:"Minimized"

  • 最大化窗口（显示任务栏和标题栏）

    visibility:"Maximized"

  • 全屏显示（隐藏任务栏和标题栏）

    visibility:"FullScreen"

  • 隐藏窗口，不可见，与visible属性的效果一样

    visibility:"Hidden"

color: 窗口的背景颜色

opacity：设置窗口的透明度，取值范围是0~1.0

title：设置窗口标题

模态与非模态

  • 非模态

     modality:"NonModal"

  • 窗口级别的模态

    modality:"WindowModal"

  • 应用程序级别的模态

    modality:"ApplicationModal"

4.2.2 ApplicationWindow#

APPlicationWindow 是 Window 的派生类，使用以下语句来引入Controls模块

  import QtQuick.Controls

APPlicationWindow 有菜单栏、工具栏、状态栏等属性

4.3.3 渐变色#

QML中渐变色的类型是Gradient，Gradient中的GradientStop用来指定一个颜色值和一个位置（取值0.0~1.0）

示例如下：

Rectangle {
          width: 100
          height: 100
          gradient: Gradient {
               GradientStop {
                position: 0.0
                color: "red"
               }
               GradientStop {
                position: 1.0
                color: "blue"
               }
          }

使用rotation属性，改变渐变反向

Rectangle {
          width: 100
          height: 100
          rotation: 90 //旋转90度
          gradient: Gradient {
               GradientStop {
                position: 0.0
                color: "red"
               }
               GradientStop {
                position: 1.0
                color: "blue"
               }
          }
