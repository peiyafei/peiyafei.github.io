SQL必知必会

《SQL必知必会》学习笔记 第一章 了解SQL 1.1 数据库的基础 1.1.1 数据库 数据库（database）：保存有组织的数据的容器。 DBMS：数据库管理系统 1.1.2 表 表：某种特定类型数据的结构化清单。 表名：在同一数据库中不能使用相同的表名。 1.1.3 列和数据类型 表由列组成 列（column）：表中的字段。 数据类型：用于限制列中所存储的数据类型 1.1.4 行 行（row）：表中的记录 1.1.5 主键 主键（primary key）：用于唯一表示表中每一行 主键的特点： 唯一性 不允许为NULL值 不允许修改或更新 不能重用（不能删除之后赋予新行） 1.2 什么是SQL SQL：Structured Query Language（结构化查询语言） 第二章 检索数据 2.1 SELECT 语句 关键字(keyword)：作为SQL组成部分的保留字，不能用作为表或列的名字。 SELECT检索数据，必须至少给出两条信息：选择什么，从什么地方选择 2.2 检索检索单个列 SELECT prod_name FROM dbo.Products; 提示： 建议SQL语句结尾总是加上分号 SQL语句不区分大小写 在处理SQL语句时，所有的空格都会被忽略，SQL语句可以写成一行或多行 2.3 检索多个列 检索多个列，在多个列名之间用逗号隔开（最后一列不能加） SELECT prod_id,prod_name,prod_price FROM dbo.Products; 2.4 检索所有列 星号 * 通配符代表所有的列 SELECT * FROM dbo....

Qt简介

Qt简介 Qt 是一个1991年由Qt Company开发的跨平台C++图形用户界面应用程序开发框架。它既可以开发GUI程序，也可用于开发非GUI程序，比如控制台工具和服务器。Qt是面向对象的框架，使用特殊的代码生成扩展（称为元对象编译器(Meta Object Compiler, moc)）以及一些宏，Qt很容易扩展，并且允许真正地组件编程。
