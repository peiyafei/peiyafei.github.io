第二章 新手必须掌握的Linux命令#

2.3 常用的系统工作命令#

  1. echo 命令

    输出字符

    # echo hellworld

    输出变量值

    # echo $SHELL

  2. date 命令

    查看当前时间

    # date

    按照"年-月-日 小时：分钟：秒"的格式查看当前系统时间

    # date +"%Y-%m-%d %H:%M:%S"

    将时间设置为2017年9月1日8点30分

    # date -s "20170901 8:30:00"

    查看今天是当年中的第几天

    # date +"%j"

  3. reboot 命令

    重启操作系统（只能root管理员使用）

    # reboot

  4. poweroff 命令

    关闭系统（只能root管理员使用）

    # poweroff

  5. wget 命令

    在终端中下载网络文件

    # wget http://www.linuxprobe.com/docs/LinuxProbe.pdf

  6. ps 命令

    查看系统的进程状态

    # ps -aux

  7. top 命令

    动态的监视进程活动与系统负载

    # top

  8. pidof 命令

    查询某个指定进程的PID值

    # pidof sshd

  9. kill 命令

    终止某个指定的PID的服务进程

    # kill 2156

  10. killall 命令

    终止某个指定名称的服务所对应的全部进程

    # killall sshd

2.4 系统状态监测命令#

  1. ifconfig 命令

    获取网卡配置与网络状态等信息

    # ifconfig

  2. uname 命令

    查看系统内核与系统版本

    # uname -a

    查看系统版本信息

    # cat /etc/redhat-release

  3. uptime 命令

    查看系统负载信息

    # uptime

  4. free 命令

    显示当前系统中内存的使用量

    # free -h

  5. who 命令

    查看当前登入主机的用户终端信息

    # who

  6. last 命令

    查看系统所有的登录记录

    # date

  7. history 命令

    显示历史执行过的命令

    # history

    清空执行过的历史命令

    # history -c

  8. sosreport 命令

    收集系统配置及架构信息并输出诊断文档

    # sosreport

2.5 工作目录切换命令#

  1. pwd 命令

    显示用户当前所处的工作目录

    # pwd

  2. cd 命令

    切换工作路径

    # cd /etc/

    返回到上一次的目录

    # cd -

    切换到用户的家目录

    # cd ~

  3. ls 命令

    显示目录中的文件信息，参数a表示查看全部文件包含隐藏文件，参数 l 显示文件的详细信息

    # ls -al

2.6 文本文件编辑命令#

  1. cat 命令

    查看纯文本文件（内容较少的），参数n显示行号

    # cat -n initial-setup-ks.cfg

  2. more 命令

    查看纯文本文件（内容较多的），用空格或者回车翻页

    #  more initial-setup-ks.cfg

  3. head 命令

    查看纯文本文件的前N行

    # head -n 20 initial-setup-ks.cfg

  4. tail 命令

    查看纯文本文件的后N行或持续刷新内容

    # tail -n 20 /var/log/messages

  5. tr 命令

    替换文本文件中的字符

    # cat initial-setup-ks.cfg | tr [a-z] [A-Z]

  6. wc 命令

    统计指定文本的行数、字数、字节数

    # wc -l /etc/passwd

  7. stat 命令

    查看文件的具体存储信息和时间等信息

    # wc -l /etc/passwd

  8. cut 命令

    按列提取文本字符，参数 -d 设置指明间隔符合，参数 -f 指定列数

    # cut -d: -f1 /etc/passwd

  9. diff 命令

    比较多个文本文件的差异，参数 –brief 显示比较后的结果

    # diff --brief A.txt B.txt

    参数-c描述文件具体不同的内容

    # diff -c A.txt B.txt

2.7 文件目录管理命令#

  1. touch 命令

    创建空白文件

    # touch text.txt

    修改文件时间，参数a仅修改读取时间，参数m仅修改修改时间，参数-d同时修改atime和mtime

    # touch -d "2017-07-11 18:01" text.txt

  2. mkdir 命令

    创建空白的目录

    # mkdir linuxprobe

  3. cp 命令

    复制文件或者目录

    # cp AA.txt  BB.txt

  4. mv 命令

    剪切文件（不同目录）

    # cp AA.txt /home/linuxprobe/

    文件重命名（同一目录，目标文件不存在）

    # mv BB.txt  CC.txt

  5. rm 命令

    删除文件，-f参数为强制删除不询问

    # rm -f AA.txt

    -r参数用于删除目录

    # rm -rf test/

  6. dd 命令

    按照指定大小和个数的数据库来复制文件

    # dd if=/dev/zero of=560_file count=1 bs=560M

    制作光盘镜像文件

    # dd if=/dev/cdrom of=RHEL-server-7.0-x86_64-LinuxProbe.com.iso

  7. file 命令

    查看文件的类型

    # file /dev/sda

2.8 打包压缩和搜索命令#

  1. tar 命令

    压缩

    # tar -zcvf etc.tar.gz /etc/

    解压，-C参数指定解压目录

    # tar -zxvf etc.tar.gz  -C  test01/

  2. grep 命令

    （在文本中执行关键词搜索，并显示匹配结果）

    查询不允许登录系统的所有用户

    # grep /sbin/nologin  /etc/passwd

  3. find 命令

    （按照指定条件来查找文件）

    获取/etc目录中以host开头的文件列表

    # find /etc/ -name  "host*" -print

    搜索整个系统中权限包括SUID权限的所有文件

    # find / -perm -4000 -print

第三章 管道符、重定向与环境变量#

3.1 输入输出重定向#

  • 标准输入重定向（STDIN，文件描述符为 0 ）
  • 标准输出重定向（STDOUT，文件描述符为 1 ）
  • 错误输出重定向（STDERR，文件描述符为 2 ）

通过标准输出重定向将man bash命令原本要输出到屏幕的信息写入到readme.txt中

# man bash >readme.txt

覆盖写入

# echo "welconme to LinuxProbe.com" >readme.txt

追加写入

# echo "Quality linux learning materials" >>readme.txt

错误输出重定向

# ls -l linuxprobe 2> stderr.txt

输入重定向

# wc -l < readme.txt

3.2 管道命令符#

统计被限制登录用户数

# grep /sbin/nologin /etc/passwd | wc -l

自动化脚本修改密码操作

# echo "linuxprobe" | passwd --stdin root

3.3 命令行的通配符#

用通配符 *查看硬盘分区数，其*匹配零个或多个字符

# ls -l /dev/sda*

用通配符？查看硬盘分区数，其中？代表一个字符

# ls -l /dev/sda?

使用 [0-9] 匹配单个字符

# ls -l /dev/sda[0-9]

3.4 常用的转义字符#

  • 反斜杠（\）
  • 单引号('')
  • 双引号("")
  • 反引号(``)

用双引号保留其中的字符串和变量信息

# price=5
# echo "price is $price"

使用\保持后面紧接着的的变量为单纯的字符

# echo "price is \$$price"

命令执行后，返回结果

# echo `uname -a`

3.5 重要的环境变量#

用type判断命令是内部命令还是外部命令

# type cd

检查rm命令是否有别名

# alias rm

定义命令别名

# unalias rm

取消命令别名

# unalias ls

显示环境变量

# $PATH

增加环境变量

PATH=$PATH:/root/bin

4.1.4 配置Yum软件仓库#

  1. 在/etc/yum.repos.d目录中创建rhel7.repo文件

    # vim /etc/yum.repos.d/rhel7.repo

  2. 新建/media/cdrom目录

    # mkdir /media/cdrom/

  3. 编辑配置文件

    [rhel7]
name=rhel7
baseurl=file:///media/cdrom
enabled=1
gpgcheck=0

  4. 手动挂载

    # mount  /dev/cdrom /media/cdrom/

  5. 设置开机自动挂载

    # vim /etc/fstab

    追加如下代码

    /dev/cdrom /media/cdrom iso9660 defaults 0 0

  6. 安装httpd以验证仓库配置结果

    # yum install httpd -y

第七章 使用RAID与LVM磁盘阵列技术#

  • RAID（独立冗余磁盘阵列）
  • LVM（逻辑卷管理器）

RAID 0 技术，至少两块，通过硬件或软件的方式串联在一起。

RAID 1 技术，将两块以上的硬盘设备进行绑定（镜像备份）。

RAID 5 技术，将硬盘设备的数据奇偶校验信息保存到其他硬盘设备中。

RAID 10 技术（常用），至少4块硬盘来组建，是RAID 1 + RAID 0 的组合体。先组合为RAID 1 ，再实施 RAID 0 技术

mdadm 命令用于管理Linux系统中的软件的 RAID 硬盘阵列

先创建 raid ，再格式化，再挂载，再设置开机自动挂载

磁盘阵列+备份盘：RAID 5 + 备份盘
